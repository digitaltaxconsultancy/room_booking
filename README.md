# Room Booking System

A complete hotel room booking system built with Laravel featuring room management, booking functionality, user authentication, and payment integration.

## Features

- ✅ Room Management (CRUD operations)
- ✅ Booking System with date validation
- ✅ User Authentication (Admin/Customer roles)
- ✅ Payment Integration (Stripe)
- ✅ Admin Panel
- ✅ Responsive Bootstrap UI


## Local Installation
